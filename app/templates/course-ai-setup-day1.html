📋 Инструкция дня

Времени займёт 60–90 минут. Большую часть просто ждёшь, пока что-то скачается.

Инструкция разбита на 6 этапов. Делай по порядку, не перепрыгивай. После каждого этапа можешь сделать перерыв.

  • 🖥 Этап 1. Устанавливаем VS Code — ~5 мин
  • 🔐 Этап 2. Регистрируемся в Claude (с обходом блокировок для РФ) — ~20–30 мин
  • 💳 Этап 3. Оплачиваем подписку — ~10 мин
  • ⚙️ Этап 4. Ставим Claude Code и настраиваем на максимум — ~15 мин
  • 🎤 Этап 5. Ускорители: голосовой ввод + скриншоты — ~10 мин
  • 🤖 Этап 6. Запускаем агента на установку окружения — ~10–20 мин
VS Code и Claude Code — это разные программы, которые работают вместе. Через VS Code мы работаем в Claude Code.
▶ 🖥 Этап 1. Устанавливаем VS Code развернуть

VS Code — редактор кода, твоё главное окно на весь практикум. Бесплатный.

  • Зайди на code.visualstudio.com
  • Нажми Download — скачается версия под твою систему (Mac или Windows)
  • Установи как обычную программу: двойной клик → «Далее» → «Готово»
  • Запусти. Должно открыться окно редактора с тёмным фоном

Всё, VS Code готов. Идём дальше.

▶ 🔐 Этап 2. Регистрируемся в Claude развернуть

Claude — «мозг», который будет за тебя писать код. Сначала регистрируем аккаунт, потом (на следующем этапе) оплачиваем подписку. Если ты не в России или у тебя не русская симка — просто зайди на claude.ai, зарегистрируйся через Google и переходи сразу к этапу 3.

Google-аккаунт тоже должен быть на почте Gmail. Если нет — заведи.

⚠️ Что не скармливать Claude: паспортные данные, пароли от банка, персональные данные клиентов. Claude читает твои файлы — если там лежат паспорта, они «засветятся» у Anthropic. Для идей, кода, бизнес-планов — безопасно.
▶ 💳 Этап 3. Оплачиваем подписку развернуть

Без платной подписки упрёшься в лимиты через 10 минут. Платить обязательно.

Шаг 1. Выбери тариф
  • Max x5 — $100 / мес (~7 500 ₽). Нормальная работа. Одного активного проекта хватит.
🎯 Рекомендация: минимум Max x5 за $100/мес. На Pro ты сделаешь пару запросов — и встанешь в лимит на полдня.
Шаг 2. Оплата

Если карта не русская — то напрямую:

  • Зайди на claude.ai/upgrade
  • Выбери тариф
  • Оплати
▶ ⚙️ Этап 4. Ставим Claude Code и настраиваем на максимум развернуть

Теперь подключаем Claude прямо внутрь VS Code, чтобы агент мог писать код и запускать команды в твоём редакторе.

Шаг 1. Установи Node.js

Node.js нужен, чтобы Claude мог ставить остальные программы (pnpm, Git и т.д.).

  • Зайди на nodejs.org
  • Скачай версию LTS (с зелёной кнопкой)
  • Установи, везде нажимая «Далее»
Шаг 2. Установи расширение Claude Code
  • В VS Code слева в сайдбаре найди иконку Extensions (четыре кубика) или нажми Cmd / Ctrl + Shift + X
  • В поиске напиши claude
  • Выбери официальное расширение от Anthropic (с галочкой verified, миллионы установок)
  • Нажми Install
  • После установки иконка Claude появится в сайдбаре — нажми на неё
  • VS Code откроет браузер — авторизуйся в Claude-аккаунте
  • Вернись в VS Code — в сайдбаре теперь у тебя чат с Claude Code

Если через Extensions не пошло — резервный путь через терминал. Открой его в VS Code (View → Terminal в верхнем меню) и вставь команду:

npm install -g @anthropic-ai/claude-code
Шаг 3. Настрой Claude на максимум
🚨 Это самый важный шаг. Без него Claude работает вполсилы — туповато и без учёта контекста.

Нажми шестерёнку (⚙️) в панели Claude Code → выставь все 5 настроек:

  • Permission Mode — Bypass Permissions. Агент не спрашивает перед каждой командой.
  • Default Mode — Bypass Permissions. Каждый новый чат открывается в этом режиме.
  • Effort — Max. Думает глубже, делает качественнее.
  • Thinking — On. Включает «размышление вслух» перед ответом.
  • Model — Claude Opus 4.7. Самая умная модель, 1 млн токенов контекста.
Если в меню нет Opus 4.7 или Bypass Permissions

Значит у тебя старая версия расширения. Фикс:

  • Extensions → Claude → правая кнопка → Reload (или Update)
  • Закрой VS Code полностью (Cmd / Ctrl + Q — не просто сверни)
  • Открой заново
  • Настройки появятся
▶ 🎤 Этап 5. Ускорители: голосовой ввод + скриншоты развернуть

Эти два инструмента ускорят тебя в 3–5 раз. Без них практикум превратится в боль.

Шаг 1. Голосовой ввод

Надиктовать голосом в 3 раза быстрее, чем набрать клавиатурой. Встроенная диктовка Claude с русским работает плохо — ставим отдельный инструмент.

Mac — любой из двух на выбор:

  • Wispr Flow — wisprflow.ai
  • Super Whisper — superwhisper.com

Windows:

  • Wispr Flow — wisprflow.ai

Что сделать после установки:

  • Скачай и установи с сайта
  • Дай программе доступ к микрофону (попросит при первом запуске)
  • Настрой горячую клавишу: зажал → диктуешь → отпустил → текст вставляется в активное поле
  • Проверка: открой Claude Code, нажми горячую клавишу, скажи «всем привет, проверка связи» — текст должен появиться в поле ввода

Если пока не готов платить — бесплатные встроенные:

  • Mac: System Settings → Keyboard → Dictation → On
  • Windows: зажми Win + H в любом окне

Качество ощутимо хуже, но работает.

Шаг 2. Скриншоты в буфер обмена

Чтобы показать Claude ошибку, дизайн или интерфейс — нужен скриншот, который улетает сразу в буфер обмена (не сохраняется на рабочий стол). Дальше Cmd / Ctrl + V в чат.

Mac. Горячая клавиша Cmd + Shift + Ctrl + 4 — скриншот выделенной области сразу в буфер. Если неудобная — переназначь:

  • System Settings → Keyboard → Keyboard Shortcuts → Screenshots
  • Найди строку Copy picture of selected area to the clipboard
  • Кликни на текущую комбинацию и нажми свою (например, Cmd + Shift + 4)

Windows. Win + Shift + S — открывает Snipping Tool. Выделяешь область → она в буфере.

Проверка:

  • Сделай скриншот любой области экрана через горячую клавишу
  • Открой Claude Code
  • В поле ввода нажми Cmd / Ctrl + V
  • Скриншот должен появиться в чате

Если появился — всё работает. Если нет — перепроверь, что используешь именно клавишу «в буфер обмена», а не «на рабочий стол».

▶ 🤖 Этап 6. Запускаем агента на установку окружения развернуть

Самый классный этап — дальше за тебя всё делает Claude. Твоя задача — только подтверждать команды.

Шаг 1. Скопируй промпт и вставь в Claude

Скопируй промпт ниже одной кнопкой, вставь в чат Claude Code, нажми Enter.

Промпт для агента
Я полный новичок в программировании, это мой первый опыт. Я прохожу практикум. Моя задача сегодня — подготовить компьютер к разработке веб-сайтов на Next.js. Пожалуйста, сделай для меня следующее: 1. Проверь, какая у меня операционная система (Mac или Windows). 2. Проверь, что уже установлено: Node.js, pnpm, Git. Покажи версии того, что нашёл. 3. Установи то, чего не хватает: - pnpm (через npm install -g pnpm) - Git (если его нет) 4. После установки проверь ещё раз: node -v, pnpm -v, git --version. 5. Подскажи и помоги установить расширения VS Code для разработки на Next.js: ESLint, Prettier, Tailwind CSS IntelliSense, Prisma. Важные правила: - Перед каждой командой объясняй простыми словами, что собираешься сделать и зачем. - Жди моего разрешения перед запуском каждой команды. - Если что-то пойдёт не так — объясни ошибку человеческим языком и предложи решение. - В конце скажи: «Окружение готово, можно начинать практикум».

Что этот промпт сделает:

  • Поймёт твою систему — Mac или Windows, подберёт правильные команды
  • Проверит, что уже стоит — чтобы не ставить дважды
  • Установит pnpm — менеджер библиотек, скачивает кусочки проектов
  • Установит Git — система обмена кодом, нужна с Дня 3
  • Подскажет расширения VS Code — подсветка ошибок, автоформат, подсказки
Всё — твой компьютер и Claude готовы к практикуму. Если не настроишь — не страшно, увидишь на мастер-классе, как это сильно помогает экономить время и деньги.
